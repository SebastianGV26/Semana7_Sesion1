Mobile First

Es la vista mobile de la página, se maqueta hacía abajo, aqui no existe columna derecha o izquierda. SE RECOMIENDA USAR PRIMERO EL MOBILE (no tiene un dimensión definida) en mobile first viene su dimensión de menos a más y se usa la propiedad min-width.

Desktop Fist

Es la vista desktop de la página, aqui se pueden poner columnas y mostrar elementos, como cards, una al lado de la otra. Se usa el max-width y se va de más a menos en la dimensión. Width mínimo que se puede dar es 320px.

Content First

Se puede usar en ambos (mobile o desktop first) pero está orientado al contenido,

Mobile First

Se desea realizar la vista de un formulario para el registro de reservas de una empresa de buffet, pero solo se cuenta con una tablet para realizar la reserva.

Desktop Fist

Una empresa brinda servicios y para atraer clientes, decidieron hacer una web interactiva para que los potenciales clientes accedan, la empresa desea por el momento la vista web, pero no descarta hacerlo responsive

Content First

Un emprendedor busca la creacion de su sitio web, pero en primera instancia quiere ver como se vería el contenido dentro de su página, por ello, solicita al desarrollador incluir todos los elementos solicitados y mostrarle el avance para luego dimensionarlo.
